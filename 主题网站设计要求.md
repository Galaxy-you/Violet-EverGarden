### 设计目标
* 掌握从 “页面结构搭建” 到 “视觉风格落地” 再到 “轻交互实现” 应用能力
* 要求网站既具备美观的视觉效果与个性化创意，又有真实的内容
* 综合性、实用性、创意性，个性化
* 以用户需求为核心、以落地实用为目标
* 4-6 页开发体量，工作量20-30h

### 整体要求
* 完成4-6 页主题式静态独立网站，页面间逻辑连贯，形成完整的内容闭环
* 适配新媒体多端视觉体验 
* 技术栈要求：基于 HTML5+CSS3（必做）+JavaScript（适量）开发，可使用轻量 CSS 框架,但禁止使用成熟建站模板直接修改。 
* 主题鲜明且贴合新媒体设计趋势：
* 网站主题需聚焦一个具体细分领域（不要大而全），视觉风格与主题高度契合
* 体现 “视觉吸引力 + 内容可读性 + 交互友好性” 核心特点。

### 页面设计要求
* 页面结构：需包含核心必选页面+个性化选选页面，结构清晰、导航便捷，无死链、无页面跳转混乱问题
* 风格统一：全站视觉风格（色彩、字体、版式、图标）* 高度统一，无页面风格割裂问题
* 视觉元素创意：包含原创 / 个性化视觉元素（如原创 LOGO、主题图标、手绘插画、个性化横幅 banner）
* 细节设计：包含设计常用细节（如 hover 悬停效果、按钮状态变化、图片懒加载视觉提示、页面加载过渡效果），表单、按钮、输入框等交互元素设计美观，与全站风格统一.

### 加分项
* 实现响应式布局（适配电脑 / 平板 / 手机三端）；
* 代码做高级优化（如 CSS 变量（结合 JS 实现动态主题切换）、JS 函数封装（可复用，低耦合）、图片压缩优化、页面加载速度优化等）。
