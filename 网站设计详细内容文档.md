# “紫罗兰永恒花园”主题网站设计详细内容文档

## 1. 文档目的

本文件用于在正式设计与开发前，先完成网站内容层面的定义与组织。  
目标是让后续页面实现可以直接按本文件落地，减少反复改版。

---

## 2. 项目定位与内容策略

### 2.1 网站定位
- 网站类型：二次创作主题展示站（静态站）
- 主题核心：围绕“信件、记忆、情感修复”构建叙事型浏览体验
- 内容来源：以本地“参考资源”中的官网图片与文本为主体，补充必要原创导语

### 2.2 目标用户
- A类：看过作品、希望回顾角色与剧情脉络的用户
- B类：刚接触作品、希望快速理解世界观与观影顺序的用户
- C类：关注视觉、音乐与制作信息的内容型用户

### 2.3 内容目标
- 让用户在 3 分钟内理解作品主题与情感基调
- 让用户在 5-8 分钟内完成“世界观-角色-故事-音乐-制作”完整浏览闭环
- 提供可收藏的高质量图文内容，不做资讯站高频更新逻辑

### 2.4 叙事主线
- 主线句：从“不会表达情感”到“理解爱与告别”
- 叙事顺序：世界背景 -> 人物关系 -> 关键故事 -> 音乐情绪 -> 制作与外传延展

---

## 3. 全站信息架构（6页）

## 3.1 站点地图
1. 首页 `index.html`
2. 世界观 `pages/world.html`
3. 角色 `pages/character.html`
4. 故事 `pages/story.html`
5. 音乐 `pages/music.html`
6. 制作与外传 `pages/staff-side-story.html`

### 3.2 导航与跳转逻辑
- 顶部主导航：首页 / 世界观 / 角色 / 故事 / 音乐 / 制作与外传
- 页底统一导航：上一页 / 下一页 / 返回顶部
- 内容闭环建议：
  - 首页 -> 世界观 -> 角色 -> 故事 -> 音乐 -> 制作与外传 -> 首页

### 3.3 统一页面模块
- 全页通用：站点头部、面包屑、页脚版权
- 每页开头：主标题、英文副标题、引导文案（40-80字）
- 每页结尾：下一页入口卡片（保持浏览连续性）

## 4. 页面级内容详案

## 4.1 首页（Home）

### 页面目标
- 快速建立气质与主题记忆点
- 提供 5 个内容入口（世界观/角色/故事/音乐/制作）

### 内容区块
1. Hero主视觉区  
- 内容：主视觉图 + 主题标题 + 核心句  
- 标题建议：紫罗兰永恒花园  
- 副句建议：把无法言说的心意，写成能被抵达的信

2. 主题导语区  
- 内容：2-3 段简述，介绍作品核心命题（爱、失去、成长、表达）

3. 分流入口区（5卡片）
- 世界观 / 角色 / 故事 / 音乐 / 制作与外传

4. 关键语录区（可选）
- 每条 1 句，2-4 条，增强情感氛围

5. 页尾版权区
- 标注素材来源与非商用学习用途说明

### 素材复用建议
- 首选图：
  - `参考资源/TV series/TVアニメ...公式サイト_files/keyvisual_slide01.jpg`
  - `.../keyvisual_slide04.jpg`
  - `参考资源/紫罗兰永恒花园外传 剧场版/.../keyvisual01.webp`

### 需补充原创内容
- 首页导语（中文）
- 栏目一句话说明（每个入口 15-25 字）

## 4.4 故事页（Story）

### 页面目标
- 建立清晰的观看理解路径，不剧透过度但保留情感张力

### 内容区块
1. 故事总述

2. 时间线/章节线（推荐）

- 阶段三：告别与理解
- 阶段四：继续前行（连接剧场版/外传）

3. 剧场版与外传补充
- 各自 1 个子区块
- 字段：作品名 / 核心命题 / 情绪关键词 / 推荐观看顺序

4. 关键场景图文

## 6. 内容规范（写作与呈现）

### 6.1 文案语气
- 气质：克制、温柔、叙事感
- 句式：短句优先，避免口号化
- 术语：首次出现可“中文 + 原文”并列

### 6.2 文案长度建议
- 页面导语：40-100 字
- 卡片描述：20-60 字
- 正文段落：80-180 字
- 辅助说明：15-30 字

### 6.3 命名与一致性
- 人名采用统一中文译名
- 专有名词首次保留日文原名（可选）
- 时间、价格、发行信息按同一格式书写

### 6.4 图片与版权说明
- 页脚统一说明：
  - 素材来源于京都动画官方公开页面截图资源
  - 本站仅用于课程学习与非商业展示

## 8. 内容生产顺序（建议执行）

1. 先做“全站标题体系与导航文案”
2. 再做“故事页时间线 + 角色页卡片”
3. 然后完成“世界观与音乐页补文”
4. 最后写“首页导语与结尾收束”
5. 上线前统一做一次术语和口吻校对

---

## 9. 内容验收清单（上线前）

- 是否满足 4-6 页闭环结构
- 是否每页都有明确主题与导语
- 是否所有导航链接可达且无死链
- 是否核心图片均来自已授权学习素材目录
- 是否新增文案无明显重复与逻辑冲突
- 是否移动端首屏可读、段落不过长
- 是否已经加入版权与非商用声明
